# Task 06 — Rename a placeholder

**Commit type:** `refactor`

## What to do

Create `submissions/<your-username>/task-06.md` with this exact content:

```markdown
# Task 06

The user's name is PLACEHOLDER_NAME.
The user's favorite language is PLACEHOLDER_LANG.
The user is from PLACEHOLDER_CITY.
```

Then **replace every `PLACEHOLDER_*`** with real values about you. Do it in one commit.

## Why `refactor` this time?

`refactor` is for changes that don't add or fix behavior — they clean up what's already there. Replacing placeholders fits.

## Done when

- [ ] All three placeholders replaced
- [ ] Commit type is `refactor`, not `feat`
- [ ] Pushed to your branch
