# Task 08 - Write a one-sentence bio

**Commit type:** `feat`

## What to do

Create `submissions/<your-username>/task-08.md` with a single sentence that could go on your LinkedIn headline or GitHub profile.

Template:

```markdown
# Task 08 - Bio

<ONE SENTENCE - who you are, what you're doing, what you're working toward>
```

Example: "Former teacher turned aspiring full-stack developer, currently learning JavaScript and building small web projects at LaGuardia Community College."

## Done when

- [ ] Exactly one sentence
- [ ] It's specific (not "I like to code")
- [ ] Pushed to your branch
